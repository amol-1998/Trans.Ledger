***Follow the Steps to Get the Program Up and Running***

1. In command_prompt/powershell/git_bash run command "python blockchain_mynode.py".
2. Open any browser and type in the URL "http://127.0.0.1:5001" or "http://0.0.0.0:5001".
3. Login at the homepage (All necessary details are given there).
4. After that perform any of the given functions and enjoy!!!

(To decentralize the Blockchain program, run the program on different hosts and connect them to each other using the "Connect Node" functionality provided.)
